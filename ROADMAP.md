# Roadmap

Implementation should not begin until the v0.1 spec is reviewed and approved.

## Ordered phases

0. **Phase 0: Repository setup and project framing.**
1. **Phase 1: OpenIdentity v0.1 specification PR.**
2. **Phase 2: Example manifests and schema validation.**
3. **Phase 3: JSON Schema or YAML schema extraction.**
4. **Phase 4: CLI validator for `openidentity.md`.**
5. **Phase 5: Resolver for memory, MCP, A2A, wallet, and auth links.**
6. **Phase 6: Signing and verification model.**
7. **Phase 7: Reference integrations with storage providers and identity providers.**
